## 📌 Project Overview

**CreditWise** is an end-to-end Machine Learning project that predicts whether a loan application should be **Approved** or **Rejected** based on customer information and historical loan records.

The project demonstrates the complete Machine Learning workflow including:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Encoding
- Model Training
- Model Evaluation
- Loan Approval Prediction

---

## 🎯 Objective

The objective of this project is to automate the loan approval process by building intelligent machine learning models that can predict loan approval decisions accurately.

### Prediction Output

```text
✔ Approved (1)

✔ Rejected (0)
```

## 📈 Evaluation Metrics

The models were evaluated using the following metrics:

### Precision

```math
Precision = TP / (TP + FP)
```

Measures how many approved predictions were actually correct.

---

### Recall

```math
Recall = TP / (TP + FN)
```

Measures how many actual approved loans were correctly identified.

---

### F1 Score

```math
F1 = 2 × (Precision × Recall) / (Precision + Recall)
```

Measures the balance between Precision and Recall.
